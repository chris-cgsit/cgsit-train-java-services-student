Advanced REST Patterns: Search/Paging, Versioning, CORS, Rate Limiting.

Search & Paging

  • GET /api/products — All products
  • GET /api/products/search — Paginated search
  • Search Books by price DESC

API Versioning

  • GET /api/v1/info — Version 1 (basic)
  • GET /api/v2/info — Version 2 (extended)

Response Headers

  • Access-Control-Allow-Origin: * — CORS
  • X-RateLimit-Limit: 60 — Rate Limit
  • X-RateLimit-Remaining: NN — Requests left
Check response headers in browser DevTools (Network tab) or IntelliJ HTTP Client.
