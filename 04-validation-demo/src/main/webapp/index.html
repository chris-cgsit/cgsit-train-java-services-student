validation-demo

REST API mit Bean Validation (Jakarta Validation 3.0).

  • GET /api/products — Alle Produkte
  • Swagger UI — Interaktive API-Dokumentation
  • OpenAPI Spec — JSON Schema
Validation testen: POST mit ungültigen Daten senden (leerer Name, negativer Preis, ungültige Kategorie) — Server antwortet mit 400 + Fehlermeldungen.
Hinweis: Dieses Modul benötigt standalone-microprofile.xml für den /openapi Endpoint.
