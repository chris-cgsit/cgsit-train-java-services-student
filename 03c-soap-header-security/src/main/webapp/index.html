soap-header-security

SOAP Web Service with Header Parameters & Handler Chain Security.

  • WSDL — Service description with <soap:header> definitions
  • SecureCalculatorService — SOAP Endpoint (POST)
New compared to soap-demo:
  • apiKey — SOAP Header for authentication (validated by AuthHandler)
  • correlationId — SOAP Header for request tracking
  • LoggingHandler — logs raw SOAP XML in/out
  • AuthHandler — validates API key, sets role in MessageContext
  • WebServiceContext — service reads role for access control
  • divide() — requires admin role
API keys & roles:
admin-key-2025 → role admin (all operations)
training-key-2025 → role user (add, multiply only)
Client testen: RawSoapClient.main() oder TypedSecureClient.main() in IntelliJ starten.
