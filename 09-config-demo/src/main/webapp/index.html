config-demo

MicroProfile Config — Konfiguration per Properties, Env-Vars, System Properties.

  • GET /api/config — Alle Config-Werte anzeigen
  • GET /api/config/source/app.name — Woher kommt der Wert?
  • GET /api/config/source/smtp.host
  • GET /api/config/all-properties — Alle Properties auflisten
Test: Umgebungsvariablen setzen und WildFly neu starten: 
set APP_NAME=Production App
set APP_ENVIRONMENT=production
set SMTP_HOST=mail.prod.example.com
set FEATURE_DARKMODE=true
Dann GET /api/config aufrufen — Werte sind ueberschrieben.
Voraussetzung: standalone-microprofile.xml (MicroProfile Config Subsystem).
